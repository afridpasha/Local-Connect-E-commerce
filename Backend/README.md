# Local Connect - Backend API

Express.js backend API server for the Local Connect E-commerce platform.

## Development Commands

```bash
# Install dependencies
npm install

# Start server
npm start

# Start with nodemon (development)
npm run dev
```

## Port
- Development: http://localhost:5003
- Environment Variable: API_SERVER_PORT

## Features
- User Authentication
- Worker Management
- Ticket Management
- Order Processing
- Review System
- File Upload (Multer)
